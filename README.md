# 🦎 Эволюция: Администрирование партий
> Десктопное приложение для судейства, записи ходов и анализа партий настольной игры **«Эволюция»** (базовый набор).  
> **Не игровой клиент** – это система администрирования: валидация правил, база партий, рейтинги игроков.

---

## ✨ Возможности

- ✅ **Пошаговое ведение партии** – создание игры (2–4 игрока), фазы развития, питания, вымирания.
- ✅ **Валидация ходов** – автоматическая проверка по правилам базового набора (19 свойств, атаки хищников, кормовая база).
- ✅ **База данных партий** – SQLite сохраняет историю всех ходов, можно загрузить любую прошлую игру.
- ✅ **Статистика и рейтинги** – процент побед, средние очки, любимые комбинации свойств для каждого игрока.
- ✅ **Графический интерфейс** – наглядное отображение видов, свойств, кормовой базы и текущего счёта.
