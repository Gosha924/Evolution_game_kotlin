# 🦎 Эволюция: Администрирование партий

> Десктопное приложение для судейства, записи ходов и анализа партий настольной игры **«Эволюция»** (базовый набор).  
> **Не игровой клиент** – это система администрирования: валидация правил, база партий, рейтинги игроков.

## 🧬 Справочник свойств животных
В системе реализована проверка взаимодействия следующих свойств базового набора:

### ⚔️ Атакующие и агрессивные свойства
*   **Хищник (Predator)**: Позволяет атаковать других животных. При успехе хищник получает 2 синие фишки еды, а жертва погибает.
*   **Острое зрение (Sharp Vision)**: Позволяет хищнику игнорировать свойство жертвы «Камуфляж».
*   **Пиратство (Piracy)**: Позволяет животному один раз в ход забрать синюю фишку еды у другого животного (если оно накормлено).
*   **Паразит (Parasite)**: Вешается на животное противника. Увеличивает потребность в еде на 2 единицы. Не может быть сброшено «Отбрасыванием хвоста».

### 🛡️ Защитные свойства
*   **Мимикрия (Mimicry)**: При атаке хищника владелец жертвы может перенаправить атаку на другое свое животное (если хищник способен его съесть).
*   **Отбрасывание хвоста (Tail Loss)**: Позволяет пережить атаку, сбросив одно любое свойство жертвы. Хищник получает только 1 еду.
*   **Бег (Running)**: При атаке бросается кубик. На 4, 5, 6 — животное спасается.
*   **Ядовитое (Poisonous)**: Если хищник съедает это животное, он погибает в конце текущей фазы кормления.
*   **Водоплавающее (Swimming)**: Защищает от хищников без свойства «Водоплавающее».
*   **Большое (High Body Mass)**: Защищает от хищников без свойства «Большое».
*   **Норное (Burrowing)**: Если животное накормлено, его нельзя атаковать.
*   **Камувляж (Camouflage)**: Может быть атаковано только жищником со свойством «Острое зрение».

### 🍽️ Пищевые и метаболические свойства
*   **Жировой запас (Fat Tissue)**: Позволяет сохранять излишки еды в конце фазы кормления для использования в следующем ходу.
*   **Спячка (Hibernation)**: Позволяет животному не питаться в текущий ход (доступно раз в две фазы кормления).
*   **Падальщик (Scavenger)**: Получает 1 синюю фишку еды каждый раз, когда любое другое животное в игре погибает.

### 🔗 Парные свойства (Взаимодействие)
*   **Сотрудничество (Cooperation)**: Когда это животное получает еду, связанное с ним животное берет 1 еду из запаса.
*   **Взаимодействие (Communication)**: Когда это животное берет еду из общего запаса, связанное с ним животное также получает 1 еду.
*   **Симбиоз (Symbiosis)**: Симбионт не может быть атакован, пока животное-хозяин живо.
